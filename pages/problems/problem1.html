Problem 1

If you are block walking to some entry in Row $N$ you will need to pass through exactly one entry in any Row $K$ with $K\le{N}$. Be careful, though, because it is not possible to travel from all entries of Row K to your destination. Let $$X=\binom{87}{15}\binom{13}{13}+\binom{87}{16}\binom{13}{12}+...+\binom{87}{28}\binom{13}{0}$$ If $X=\dbinom{n}{k}$ where $n-k=72$, what is $n*k$?
